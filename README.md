# lava-lamp-simulator
hello my name is kenji you can call me ken. i make this website for someone to focus in something but needed to know what time, its a great use too if you want to relax bcs of our cozy music (its not copyrighted)
the key system is 
   1. Autonomous blob movement  — Lissajous + wander steering
   2. Metaball merge            — CSS blur+contrast (in style.css)
   3. SVG feTurbulence warp     — edges ripple like liquid
   4. Per-ball color lerp       — no flicker, smooth transitions
   5. Organic radius wobble     — blobs breathe
   6. Live fancy clock          — IM Fell English serif
there is a music button too, when it pressed it'll show the volume control from 0% up to 100%, how it work? simple just press it once. when the logo is orange, that means its on, and when the logo is white, its means off.
